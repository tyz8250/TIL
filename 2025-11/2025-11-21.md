### 1. Git が untracked のままになる原因

- 中身が空のファイルは Git が無視することがある

### 2. Git push ができなかった原因

- `git push -u remote origin main` は誤り
- 正しくは `git push -u origin main`
- remote はキーワードではなく "origin" のこと

📘 Terminal Navigation Cheat Sheet (TIL 用)

🔹 基本移動

cd <directory> # フォルダに移動
cd .. # 1 つ上の階層へ戻る
cd ../.. # 2 つ上の階層へ戻る
cd # ホームディレクトリへ戻る

🔹 最速で戻る・行き来するテクニック

■ 直前のディレクトリに戻る（最強）
cd -
• 直前のフォルダにワープ
• もう一度 cd - すると元の場所に戻る（トグル式）
🔹 絶対パスで瞬間移動
cd ~/TIL/2025-11
どこにいても一発で移動できる。

🔹 相対パス技（今の階層が基準）
cd ..
cd ../2025-11
cd ../../some-folder

🔹 補完（必須）
cd 20[TAB]
フォルダ名が自動補完されて高速移動できる。

🔹 pushd / popd（上級者向け）

複雑な移動を “戻れるように記録しながら” 行うコマンド。

pushd <directory> # 移動＋現在地を保存
popd # 保存していた場所に戻る

例：
pushd ../ai-agent

# ai-agent で作業

popd # 元いた場所へ戻る

✨ 一番使うのはこれだけ

cd -
cd ..
