### 2025-11-24 TIL

✅ ioutil は Go 1.16 で非推奨になっていた

今日、Go のコードを実行したときに以下の警告が出た：

"io/ioutil" is deprecated

調べたところ、Go 1.16 以降では ioutil パッケージは非推奨になっており、
代わりに os と io パッケージで同じことができるようになった。

⸻
✅ 修正した内容： saveToFile

ioutil.WriteFile を使っていた部分を os.WriteFile に変更。

```go
func (d deck) saveToFile(filename string) error {
return os.WriteFile(filename, []byte(d.toString()), 0666)
}
```

• 0666 … ファイルの権限（読み書き OK）
• []byte(...) … 文字列をバイト列にして書き込む
