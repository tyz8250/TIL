### 2025-11-24 TIL

- Go の array と slice の違いを学んだ。
  - array はサイズ固定。
  - slice はあとから append で増やせる。
- range を使った for 文の書き方を理解した。
- append の結果は再代入しないと反映されない点がポイント。

- Go の `_`（ブランク識別子）は「使わない変数を捨てる」ためのもの。
- for range では index と value の 2 つが返るため、
  不要な値を `_` に置き換えることで unused variable エラーを防げる。

- Go は複数の返り値を返せる。
- deal(d, handSize) は 2 つの deck を返す関数。
- d[:handSize] は「最初の handSize 枚」
- d[handSize:] は「残り全部」
- これで “手札” と “残り山札” を同時に返している。
