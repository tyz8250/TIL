このコードについて

```go
func TestNewDeck(t *testing.T){
	d := newDeck()

	if len(d) != 2000 {
		t.Errorf("Expected deck length of 16, but got %v", len(d))
	}

}
```

1.t \*testing.T の t は何者？

t は テスト専用の“ハンドラー”オブジェクト。
正式には：
テスト中にエラーを出したり、ログを書いたりするための“テスト管理用の構造体”

testing.T は構造体で、\*testing.T はそのポインタ。
テスト関数では、Go が自動でこの t を渡してくれる

```go
func TestSomething(t *testing.T) { ... }
```

2. t.Errorf("Expected ..., got %v", len(d))

✔ %v の意味
%v は 値をそのままフォーマットして表示する という意味。
• %d → 数字（整数）
• %s → 文字列
• %f → 小数
• %v → なんでも OK（最も一般的）
