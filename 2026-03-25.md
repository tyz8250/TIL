SQLiteのドライバをいれる
```bash
go get modernc.org/sqlite
```

```go
import (
    "database/sql"
    _ "modernc.org/sqlite"
)
```

sql.Open("sqlite", "note.db") は
「SQLiteの note.db ファイルを開いて使えるようにする」
```go
var db *sql.DB

func main() {
	var err error
	db, err = sql.Open("sqlite", "./notes.db")
	// DBを開くのに失敗した場合、プログラムを終了する
	if err != nil {
		panic(err)
	}
	defer db.Close()

```

#### noteテーブルを作る

現在はDBを開いただけでまだ保存先の箱がない

sql.Open(...) のあとに、db.Exec(...) をかく
```go
query := `
CREATE TABLE IF NOT EXISTS notes (
	id INTEGER PRIMARY KEY AUTOINCREMENT,
	title TEXT NOT NULL,
	content TEXT NOT NULL,
	created_at TEXT NOT NULL,
	updated_at TEXT NOT NULL
);
`

_, err = db.Exec(query) //テーブルを作成
if err != nil {
	panic(err)
}
```
