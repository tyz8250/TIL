## 仮説（自分の理解）
- 密結合は依存が強く、テストできない。別のデータソースがつかえない。
例
```go
func ProcessFile(f *os.File) {
// file専用
}
```

- なぜ疎結合なのか
```go
func Process(r io.Reader){
	// Readerなら何でもOK
	// 関数はio.Readerしかしらない-->具体型に依存していない
}
```

密結合
```bash
Process → os.File
```

疎結合
```bash
Process → Reader(interface)
             ↑
     os.File / Buffer / Network
```
