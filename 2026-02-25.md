## 仮説（自分の理解）
- ポインタは低レベルの概念である。pythonやjavascrptではみたことがない。
- panic防止のnilチェック
```go
  *val = *val * 20 // ポインタが指す先の値を書き換える操作
  
  var p *int // nilポインタ
  //nilを参照した場合
  `panic: runtime error: invalid memory address or nil pointer dereference`
  
  // 対策として、nilチェック
  if val == nil{
  return
  }
  
  *val = *val * 20 
```
- & は『変数が保存されているメモリアドレス』を取得する演算子。ポインタは値ではなく『場所』を扱う概念であり、データの共有、変更、効率化に使われる。
```go
x := 10
p := &x // pはxの場所を持つ
*p = 20 // 場所の中身を書き換える
```
