Write インターフェイスの定義

```go
type Writer interface {
    Write([]byte) (int, error)
}
```

Write 関数の役割:

バイトスライスを入力として受け取る
データを何らかの出力チャネル（ファイル、端末など）に送信
整数（書き込んだバイト数）とエラーを返す
Read との対比:

Read([]byte) (int, error): データを外部から取得してスライスに注入
Write([]byte) (int, error): スライスからデータを取得して外部に出力
